# Web

This package contains the public website and the browser input client for IsTyping.

## Routes

- `/`: marketing homepage for downloads, onboarding, FAQ, and project positioning
- `/input`: phone-friendly input experience for joining a room and sending input to desktop

## Run Locally

```bash
cd web
npm install
npm run dev
```

The development server runs on port `3001`.

## Build

```bash
cd web
npm run build
```

## Notes

- The homepage is designed for public alpha launch and points Windows downloads to GitHub Releases
- The input page restores the last successful room from URL or local storage when possible
- QR codes generated by the desktop app point to `/input?room=<roomId>`
